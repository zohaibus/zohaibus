### 🔧 What I'm Building

**LocalOffice** is a suite of single-file, local-first productivity tools. Each app is compressed into a single HTML file with zero external dependencies. No install. No cloud. Your data stays on your disk.

* **[LocalSheets](https://github.com/zohaibus/localsheets)**: Spreadsheet with a local AI panel that targets native localhost Ollama nodes. Built with a structured JSON-patch mutation engine to handle robotics telemetry, log triage, and edge ML workflows.
    * 👉 [**Launch LocalSheets Live**](https://zohaibus.github.io/localsheets/localsheets.html)
* **[DeckBuilder](https://github.com/zohaibus/deckbuilder)**: A presentation framework engineered entirely inside a single HTML file. Generate, layout, and ship production decks locally without ever touching PowerPoint or cloud suites.
    * 👉 [**Launch DeckBuilder Live**](https://zohaibus.github.io/deckbuilder/editor.html)
* **[Context Protocol](https://github.com/zohaibus/context-protocol)**: A human-first protocol for stateful LLM context and memory management. Plain text, Git-versionable, model-agnostic, and completely offline.
* **LocalPlan**: A single-file personal planner engineered to survive software trends. (In active development)

All tools are MIT licensed. Fork them, modify them, own your compute.
